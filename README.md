# 파이썬 프로그래밍 실습 (Python-10)

## 5주차 과제: 딕셔너리 (Dictionary)
`파이썬실습_#5.pdf` (p.42-43)에 포함된 딕셔너리 활용 과제입니다.

### ✍️ 과제 #25
> 다음과 같이 표준입력으로 키와 해당 값들을 각각 입력받은 후, dictionary로 만들고 출력

- **주요 사용 개념:**
    - `input().split()`: (중간고사 범위) 표준 입력을 공백 기준으로 나눠 리스트로 만듭니다.
    - [cite_start]`dict(zip(keys, values))`: (5주차 8p) [cite: 734-738] 키 리스트와 값 리스트를 `zip()`으로 묶어 딕셔너리를 생성합니다.

---

### ✍️ 과제 #26
> `park = { ... }`의 dictionary을 선언하고, english 점수와 science 점수를 출력

- **주요 사용 개념:**
    - [cite_start]`딕셔너리[키]`: (5주차 11p) [cite: 761-771] 특정 키(`'english'`, `'science'`)를 지정하여 해당 값을 조회합니다.

---

### ✍️ 과제 #27
> `kim = { ... }`의 dictionary을 선언하고, 모든 과목을 100점으로 수정후 출력

- **주요 사용 개념:**
    - [cite_start]`for key in 딕셔너리.keys()`: (5주차 32p) [cite: 1181-1189] 딕셔너리의 모든 키를 순회합니다.
    - [cite_start]`딕셔너리[key] = 100`: (5주차 12p) [cite: 775-797] 반복문 내부에서 각 키의 값을 100으로 덮어씁니다.

---

### ✍️ 과제 #28
> `lee = { ... }`의 dictionary을 선언하고, english의 key가 있는지 체크하고 있으면, 해당 key-value을 삭제후 출력

- **주요 사용 개념:**
    - [cite_start]`if 'key' in 딕셔너리:`: (5주차 14p) [cite: 813-819] `in` 연산자를 사용해 키의 존재 여부를 `True`/`False`로 확인합니다.
    - [cite_start]`del 딕셔너리[키]`: (5주차 23p) [cite: 1029-1037] `del` 명령어로 특정 키-값 쌍을 삭제합니다.

---

### ✍️ 과제 #29
> `lim = { ... }`의 dictionary을 선언하고, key와 value을 모두 출력

- **주요 사용 개념:**
    - [cite_start]`for key, value in 딕셔너리.items()`: (5주차 31p) [cite: 1161-1174] `.items()` 메서드를 사용해 `for`문에서 키와 값을 동시에 가져옵니다.

---

### ✍️ 과제 #30
> `choi = { ... }`의 dictionary을 선언하고, 90점 이상인 과목들의 key만 출력 (단, dictionary comprehension 사용)

- **주요 사용 개념:**
    - [cite_start]`{k: v for k, v in 딕셔D.items() if v >= 90}`: (5주차 35p) [cite: 1226-1245] `if` 조건문이 포함된 딕셔너리 컴프리헨션을 사용하여 90점 이상인 항목만 필터링합니다.
    - [cite_start]`.keys()`: (5주차 32p) [cite: 1181-1189] 필터링된 새 딕셔너리에서 키만 추출하여 출력합니다.

---

### ✍️ 과제 #31
> `yoo = { ... }`의 dictionary을 선언하고, 4과목의 평균을 출력

- **주요 사용 개념:**
    - [cite_start]`딕셔너리.values()`: (5주차 33p) [cite: 1193-1204] 딕셔너리의 모든 값(점수)들만 가져옵니다.
    - [cite_start]`sum()` / `len()`: (5주차 15p [cite: 840-844] + 기본 함수) 값들의 총합을 개수로 나누어 평균을 계산합니다.
