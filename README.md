### ✍️ 과제 #25
> 다음과 같이 표준입력으로 키와 해당 값들을 각각 입력받은 후, dictionary로 만들고 출력

- **주요 사용 개념:**
    - `input().split()`: (중간고사 범위) 표준 입력을 공백 기준으로 나눠 리스트로 만듭니다.
    - `dict(zip(keys, values))`: (5주차 8p) 키 리스트와 값 리스트를 `zip()`으로 묶어 딕셔너리를 생성합니다.

---

### ✍️ 과제 #26
> `park = { ... }`의 dictionary을 선언하고, english 점수와 science 점수를 출력

- **주요 사용 개념:**
    - `딕셔너리[키]`: (5주차 11p) 특정 키(`'english'`, `'science'`)를 지정하여 해당 값을 조회합니다.

---

### ✍️ 과제 #27
> `kim = { ... }`의 dictionary을 선언하고, 모든 과목을 100점으로 수정후 출력

- **주요 사용 개념:**
    - `for key in 딕셔너리.keys()`: (5주차 32p) 딕셔너리의 모든 키를 순회합니다.
    - `딕셔너리[key] = 100`: (5주차 12p)
